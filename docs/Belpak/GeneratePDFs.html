Instructions for Generating PDFs

  1. Click the "Open Document" button for the brief you want to generate
  2. Once the document opens in a new tab, press Ctrl+P (Windows) or Cmd+P (Mac)
  3. In the print dialog:
    • Set Destination to "Save as PDF"
    • Set Margins to "Default" or "Normal"
    • Enable "Background graphics" option
    • Set Paper size to "Letter"
  4. Click Save and choose your destination folder
